# Array va String Vazifalari

Ushbu loyiha C# dasturlash tilida massivlar (Array) va satrlar (String) bilan ishlash bo'yicha 10 ta amaliy vazifani o'z ichiga oladi.

##  Vazifalar Ro'yxati:
1.  **1D Array:** 10 ta sonni qabul qilish va yig'indisini hisoblash.
2.  **Min/Max:** Massiv ichidagi eng katta va eng kichik qiymatni topish.
3.  **Juft va Toq:** Sonlarni guruhlarga ajratib chiqarish.
4.  **O'rtacha qiymat:** Elementlarning o'rtacha arifmetigini topish.
5.  **Reverse String:** Satrni teskari aylantirish.
6.  **Unli/Undosh:** Harflar turini hisoblash.
7.  **Palindrom:** So'zni palindromga tekshirish.
8.  **StringBuilder:** So'zlarni bitta gapga birlashtirish.
9.  **Trim Spaces:** Bo'sh joylarni olib tashlash.
10. **Array + StringBuilder:** Ismlar ro'yxatini formatlab chiqarish.

##  Texnologiyalar
* **.NET 6/8**
* **C#**
* **System.Text (StringBuilder)**
